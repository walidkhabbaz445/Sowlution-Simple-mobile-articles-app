# Simple News App (Flutter)

##  Overview
This is a simple Flutter application that fetches and displays news articles using the **GNews API**.  
Users can:
- View top news  
- Search articles by keyword  
- See title, description, author, and image  

The UI is intentionally simple and beginner-friendly.

---

##  Features
- Fetch top N news articles  
- Search for news by keyword  
- Basic layout with safe image loading  
- Human-written, beginner-level Flutter code  

---

##  How to Run the Application (Chrome / Desktop)

Follow these exact steps:

---

### **1. Clone the Repository**
```bash
git clone https://github.com/walidkhabbaz445/Sowlution-Simple-mobile-articles-app.git
cd Sowlution-Simple-mobile-articles-app
```
### **2. Install Dependencies**
```bash
flutter pub get
```
### **3. Enable Flutter Web**
```bash
flutter config --enable-web
```
### **4. Run the App**
```bash
flutter run
```
